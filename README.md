## Install binary for yourself
### Prerequisites
```
gcc >= 13
clangd >= 18
cmake >= 3
```

### Installation
```bash
git clone https://github.com/cmaker-dev/cmaker.git
cd cmaker
cmake -DCMAKE_BUILD_TYPE=Release ./
sudo make install
```
